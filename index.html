Project

多無人飛行載具共同控制系統 Multiple drones control system

This is my university graduated project which uses Python compile KML(Keyhole Markup Language) and run system on Google Earth. The system allows user input string then would output every drone coordinate in each character. Also, the system would calculate route of each drone from one character to next character and makes sure not have a collide. There would be a moving process vedio to show the result.

此大學畢業專題使用Python編譯KML語言並在Google Earth上執行，系統允許使用者輸入字串，則會輸出每個字所需的無人機數及各無人機的座標，系統會自動計算排字路線並確保無人機移動時不會相撞，且會輸出一個移動路徑影片來使使用者觀看。

此為我大二的一個小專案，利用Python的多元套件製作的即時翻譯機，允許使用者對程式講話，程式會呈現輸入的語言及翻譯後的語言並唸出來。

迷你投籃機 Shooting Machine - Raspberry pi project for IOT

Raspberry pi as a host server connects with a servo motor to move the basket. The micro-switch detects whether the ball is into the basket or not. Using python tkinter package to build GUI.

使⽤ raspberry pi 作為主機，連接伺服⾺達來移動籃框、微動開關偵測是否投進的IOT應⽤，分為簡單模式和移動模式(籃框會移動)，使⽤ tkinter(python) 來實作GUI介⾯。
